## Browser 工具选择策略

| 场景 | 选择 | 理由 |
|---|---|---|
| 简单网页操作、OpenClaw 内置流程 | `browser` (OpenClaw 工具) | 原生集成，无需 shell |
| 复杂认证流程、auth vault | `agent-browser` | 支持 `auth login`、状态持久化 |
| 需要 diff/对比功能 | `agent-browser` | 内置 `diff snapshot/screenshot` |
| 需要 HAR 录制/网络分析 | `agent-browser` | `network har start/stop` |
| 多会话并行 | `agent-browser` | `--session-name` 隔离 |
| 远程 CDP、node 代理 | `browser` | OpenClaw 原生支持 |
| iOS Simulator 测试 | `agent-browser` | `-p ios` 支持 |
| Lightpanda 引擎（轻量） | `agent-browser` | `--engine lightpanda` |

## agent-browser 核心命令

```bash
# 导航
agent-browser open <url>
agent-browser close

# 快照（获取元素引用 @e1, @e2...）
agent-browser snapshot -i

# 交互
agent-browser click @e1
agent-browser fill @e2 "text"
agent-browser select @e3 "option"
agent-browser press Enter

# 等待
agent-browser wait @e1
agent-browser wait --load networkidle
agent-browser wait --url "**/dashboard"

# 截图/捕获
agent-browser screenshot --full
agent-browser screenshot --annotate  # 带标注的截图
agent-browser pdf output.pdf

# 认证
agent-browser auth save <name> --url <login-url> --username <user> --password-stdin
agent-browser auth login <name>

# 会话
agent-browser --session-name <name> open <url>
agent-browser session list
```
