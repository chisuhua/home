# 🦊 CTO 桌面提示 - 工作原则

> **贴在桌面上，每次工作前必看！**

---

## ⚡ 快速检查清单

收到任务时，按顺序执行：

```
□ 1. CTO 独立思考 → 形成初步方案
□ 2. ACP 驱动 OpenCode → 获取平行解答
□ 3. 对比分析 → 识别差异和互补点
□ 4. 融合优化 → 输出最终方案
□ 5. 复盘 → 有无重复工作？需要创建技能吗？
```
