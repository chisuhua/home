### 改进 1: 文档结构整理

**问题**: temp/ 中有 v1.0 + v2.0 两个版本文档，容易混淆

**解决方案**:
```
~/.openclaw/workspace/docs/workflow/
├── zcf-workflow.md              # 权威版（生效中）
├── execution-log-template.md    # 执行日志模板
├── archive/
│   ├── zcf-workflow-v1.md       # 归档 v1.0
│   └── zcf-workflow-v2-draft.md # 归档 v2.0 草案
└── skills/
    ├── zcf-architect/SKILL.md
    ├── zcf-reviewer/SKILL.md
    ├── zcf-coordinator/SKILL.md
    └── zcf-executor/SKILL.md
```

**效果**: 单一权威文档，易于查找和维护
